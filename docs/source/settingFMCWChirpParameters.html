Setting FMCW Chirp Parameters

Using the Interactive GUI

        Under the Radar Setup tab, locate the FMCW Chirp Parameters panel

 

        Enter the desired chirp parameters. A short description of each parameter:

o       Starting Frequency The starting frequency of the FMCW chirp

o     Freq Slope The slope of the linear chirp across time

o     Idle Time Duration of time before starting the chirp after the previous chirp (not included in the simulation currently, but matches TI mmWave Studio parameters)

o     TX Start Time Duration of time before the chirp starts wherein the transmitter is on (not included in the simulation currently, but matches TI mmWave Studio parameters)

o     ADC Start Time Duration of time after the transmitter is on and TX Start Time has passed wherein the chirp is active, but sample are not being taken

  Affects the useful bandwidth. As ADC Start Time increases, more bandwidth is not sampled

o       Sampling Frequency sampling frequency of the IF signal

o     Ramp End Time Duration of time after transmitter is on and TX Start Time has passed before end of the chirp

  Actual duration of time during which samples are taken is (Ramp End Time - ADC Start Time)

o     Center Frequency center frequency of the device, used to calculate

o     Maximum Bandwidth limits the allowable bandwidth to a given value

 

        As the desired chirp parameters are entered, the app updates these fields as calculated by:

o       Total Bandwidth

o       Useful Bandwidth

o          Wavelength , c is the speed of light

o       Maximum Range

o          Range Resolution

 

        Additionally, the instantaneous frequency and wavenumber are calculated as:

o       Instantaneous frequency

o       Instantaneous wavenumber

Saving and Loading Chirp Parameters

Using the Interactive GUI

        The current FMCW chirp parameters can be saved by pressing the Save button under the Save/Load FMCW Chirp Parameters panel

o   If the name given in the field to the right of the save button refers to an existing file under /application/saved/fmcwChirpParameters/, the user is prompted whether they would like to overwrite the file

        The user can load previously saved FMCW chirp parameters by pressing the Load button

o   If the file does not exist under /application/saved/fmcwChirpParameters/, the user is prompted, and the load fails
