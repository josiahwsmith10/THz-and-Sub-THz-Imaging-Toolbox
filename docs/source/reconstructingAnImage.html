Reconstructing an Image

Using Interactive GUI

        Under the Reconstruct Image tab, the user can specify the reconstruction algorithm and execute the image reconstruction process.

o   Under the Multistatic to Monostatic Conversion panel, the user can specify whether to use the multistatic-to-monostatic EPC approximation assuming the software is operating in MIMO mode

o   Under the Reconstruction Algorithm panel, the user can specify the reconstruction algorithm from the list provided.

  As the user selects the desired reconstruction algorithm, the software validates the reconstruction algorithm for use with the given array and scan type. Then, the software updates the app with helpful information under the tab group to the right relating to the imaging parameters given the various parameters entered by the user in the Reconstruction Algorithm panel

  The Use GPU check box indicates if the GPU is to be used for the image reconstruction process. Results vary.

  To learn more about the reconstruction algorithms, visit the reconstruction algorithms documentation.

  To execute the image reconstruction process, the user can press the Reconstruct Target Button

o   Under the Display Options panel, the user can choose to redisplay the figure with the option of changing the minimum point shown in dB and font size, for image sharing. Or, the user can open the reconstructed image in MATLABs Volume Viewer application

 

        Once the reconstruction process is complete, the image is displayed.
