THz and Sub-THz Imaging Toolbox and Interactive GUI
A toolbox for near-field THz simulation and image reconstruction.
More details
          This toolbox enables rapid development of near-field THz imaging radar systems by providing 
          easy access to efficient simulation tools and diverse toolset with documented API. Additionally, 
          the toolbox provides efficient generation of large near-field THz imaging scenarios useful 
          for many data-driven applications.

Utility Classes
THzWaveformParameters The chirp parameters of the FMCW radar under test, adhering loosely to the TI mmWave Studio definitions
THzAntennaArray Holds the properties and methods of the MIMO antenna array as specified by the user
THzScanner Holds the properties and methods used for simulating a THz-SAR scanning scenario in four modes: linear, rectilinear (planar), circular, or cylindrical
THzTarget Holds the properties and methods used in the target scene as specified by the user
THzImageReconstruction Holds the properties and methods used for reconstructing the image from the simulated THz scenario
Reconstruction Algorithms
XY SAR XYZ RMA 3-D XYZ range migration algorithm (RMA) over 2-D XY rectilinear SAR scan with linear MIMO array or linear uniform EPC array
XY SAR XY FFT 2-D XY FFT imaging over 2-D XY rectilinear SAR scan with linear MIMO array or linear uniform EPC array
XY SAR XYZ BPA 3-D XYZ backprojection algorithm (BPA) over 2-D XY rectilinear SAR scan with any MIMO or EPC array
XY SAR XY BPA 2-D XY backprojection algorithm (BPA) over 2-D XY rectilinear SAR scan with any MIMO or EPC array
Y SAR YZ RMA 2-D YZ range migration algorithm (RMA) over 1-D Y linear SAR scan with linear MIMO array or linear uniform EPC array
Y SAR Y FFT 1-D XY FFT imaging over 1-D Y linear SAR scan with linear MIMO array or linear uniform EPC array
Y SAR YZ BPA 2-D YZ backprojection algorithm (BPA) over 1-D Y linear SAR scan with any MIMO or EPC array
Theta-Y CSAR XYZ PFA 3-D XYZ polar formatting algorithm (PFA) over 2-D Theta-Y cylindrical SAR scan with linear MIMO array or linear uniform EPC array
Theta CSAR XZ PFA 2-D XZ polar formatting algorithm (PFA) over 1-D Theta circular SAR scan with one transceiver pair or one monostatic EPC element antenna array
Theta-Y CSAR XYZ BPA 3-D XYZ backprojection algorithm (BPA) over 2-D Theta-Y cylindrical SAR scan with any MIMO or EPC array
