Include Necessary Directories

addpath(genpath("../"))

Create the Objects

wav = THzWaveformParameters();

doc THzWaveformParameters

Set Waveform Parameters

wav.Nk = 50;
wav.f0 = 300*1e9;
wav.fS = 1000*1e3;
wav.fC = 305*1e9;
wav.B = 10e9;

Compute the Waveform Parameters

Computes the wavenumber vector etc.

wav.Compute();

Load in Saved Waveform Parameters

wav.Load();

Warning: Waveform parameters file not loaded!

Save New Waveform Parameters

wav.Nk = 79;
wav.f0 = 77*1e9;
wav.fS = 2000*1e3;
wav.fC = 79*1e9;
wav.B = 4e9;
wav.Compute();

wav.Save();

Warning: Waveform parameters file not saved!
