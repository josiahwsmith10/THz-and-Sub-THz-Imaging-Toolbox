Developing a Custom Image Reconstruction Algorithm

Using Toolbox

        Navigate to /utilities/reconstructionAlgorithms/reconstructionAlgorithmTemplate.m

o   Documentation for the template can be found here.

        To use your image reconstruction algorithm, change the property of the THzImageReconstruction object called method to the string value custom.

        To execute image reconstruction using your algorithm, call the Compute() method of the THzImageReconstruction object

Development Steps:

1.      Modify the getParameters() method of the object to set the properties of the object according to your algorithms needs

a.      Add the necessary properties at the top of the class as needed, depending on your algorithm.

 

2.      Optional: modify the verifyParameters() and verifyReconstruction() methods of the object to verify the imaging parameters and other scan parameters, respectively, before reconstruction.

a.      If these are not updated, the image reconstruction process may yield errors, but these are optional.

 

3.      Add the necessary functionality to reconstruct the image from obj.sarData and set the image to obj.imXYZ in the reconstruct() method

 

4.      Finally, modify the displayImage() method to call the necessary function for displaying the reconstructed image, or implement your own image display routine

a.      For more info, visit the documentation on displayImage3D(), displayImage2D(), or displayImage1D().
