Creating a MIMO-SAR Scan Scenario

Using Interactive GUI

        Under the Device Setup tab, locate the Scanner Setup panel

 

        The software supports 4 different scanning regimes:

o   Linear

  Scans the MIMO array across the y-axis in steps of size Y-Step Size, which can be expressed in terms of multiples of the wavelength or in millimeters using the switch.

  Requires the array to be colinear along the y-dimension. Therefore, none of the enabled elements can be offset from each other in the x-direction.

o   Rectilinear

  Scans the MIMO array across the x-y plane in steps of size X-Step Size and Y-Step Size along the x and y directions, respectively.

  Currently, the software requires the array to be colinear, although support of nonlinear MIMO arrays is coming soon.

o   Circular

      Scans the MIMO array across the dimension, simulating a rotation of the target or spotlight axial rotation of the radar around the target.

  Requires the array to use a single transceiver pair to synthesize the circular pattern.

o   Cylindrical

      Scans the MIMO array across the and y dimensions synthesizing a cylindrical aperture.

  Requires the array to be colinear

 

        The number of steps along each dimension are given by

o   Num X Steps number of steps along the x-dimension

o   Num Y Steps number of steps along the y-dimension

o     Num Steps number of steps along the angular -dimension

 

        In addition to the location, the user can specify whether to use the MIMO array or the equivalent-phase-center (EPC) virtual elements by using the switch in the lower right corner

o   When in Use MIMO Array mode, the software will simulate the beat signal from the multistatic MIMO array created by the user

o   When in the Use EPC Virtual Elements mode, the software will simulate the beat signal from a monostatic array consisting of elements at the EPC points of the MIMO array, called virtual elements

 

        As the parameters of the SAR scenario are changed by the user (or the parameters of the antenna array), the figure is automatically updated showing the SAR array synthesized by the scan.

o   It is important to note that the array is synthesized in TDM MIMO-SAR mode, meaning a signal transmitted from a single transmitter is not captured by all the receivers. Rather, only the receivers making up the antenna array at that given step in the scan.
